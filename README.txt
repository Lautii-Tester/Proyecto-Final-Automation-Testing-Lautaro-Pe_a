1. Proposito del proyecto:
Poner en práctica todo lo aprendido Curso de Automatizacion de Testing, como la interacción con elementos web, estrategias de localización, realizando pruebas de UI utilizando Selenium WebDriver ejecutado por Python. Ademas de ejecutar pruebas de API mediante la biblioteca Requests. Los sitios utilizados para el Trabajo Final Integrador fueron saucedemo.com y jsonplaceholder.typicode.com, ambas páginas web especialmente diseñadas para prácticas de testing.

2. Tecnologias utilizadas:
Python como lenguaje principal
Pytest para estructura de testing
Selenium WebDriver para automatización
Git y GitHub para control de versiones
Librería Faker para brindar credenciiales aleatorias
Librería Request para ejecutar solicitudes HTTP

3. Intrucciones de Instalación de dependencias:
Python    --> http://python.org/downloads/           --> 3.14.0 version
Pytest    --> pip install pytest                     --> 8.4.2 version
Selenium  --> pip install selenium                   --> 4.36.0 version
Pytest report-html --> pip install pytest-html
Webdriver(Edge)--> (pagina oficial, windows edge, msedgedriver)--> 142.0.3595.94 version
Git       --> https://git-scm.com/install/windows/   --> 2.51.0 versión
Librería Faker  --> pip install Faker                --> 38.2.0
Librería Requests --> pip install Requests             --> 2.32.5

4. Comando para ejecutar pruebas:
pytest --html=reporte.html
